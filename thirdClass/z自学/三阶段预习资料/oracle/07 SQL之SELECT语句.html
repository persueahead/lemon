07 SQL之SELECT语句

oracle

1 SELECT语句

SELECT标识选择哪些列。
FROM标识从哪个表中选择:

SELECT *|{[DISTINCT] column|expression[alias],...} FROM table;

编写SQL语句
- SQL 语言大小写不敏感
- SQL 可以写在一行或者多行
- 关键字不能被缩写也不能分行
- 各子句一般要分行写
- 使用缩进提高语句的可读性

2 选择全部列

select * from emp;

3 选择特定的列

select emp_id,emp_name from emp;

4 算术运算符

数字和日期使用的数学表达式

select emp_id,emp_name,emp_birthday,emp_salary+300 from emp;

5 操作符优先级

  • 乘除的优先级高于加减
  • 同一优先级运算符从左向右执行
  • 括号内的运算先执行

select emp_id,emp_name,emp_birthday,emp_salary*3+300 from emp;

6 列的别名

  • 重命名一个列。
  • 便于计算。
  • 紧跟列名，也可以在列名和别名之间加入关键字‘AS’，以便在别名中包含空格或特殊的字符并区分大小写

7 连接符

  • 把列与列，列与字符连接在一起
  • 用‘||’表示
  • 可以用来‘合成’列

8 字符串

  • 字符串可以是SELECT列表中的一个字符,数字,日期
  • 日期和字符只能在单引号中出现
  • 每当返回一行时，字符串被输出一次
