1 变量和常量概述

PL/SQL 块中可以使用变量和常量
在声明部分声明，使用前必须先声明
声明时必须指定数据类型，每行声明一个标识符
在可执行部分的 SQL 语句和过程语句中使用

2 变量和常量语法

identifier [CONSTANT] datatype [NOT NULL]   
        [:= | DEFAULT expr];

3 给变量赋值有两种方法

使用赋值语句 :=
使用 SELECT INTO 语句

4 示例

declare
    username varchar2(50);
    usersex varchar2(10);
    userid CONSTANT number:=1;--常量
    userage number;--变量
begin
    userage :=18;
    Dbms_Output.put_line('编号='||userid);
    ----赋值
    select user_name,user_sex into username,usersex 
    from user_info where user_id=userid;
    Dbms_Output.put_line('姓名='||username);
    Dbms_Output.put_line('性别='||usersex);
end;
