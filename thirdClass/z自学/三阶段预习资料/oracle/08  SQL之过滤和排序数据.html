1.3.3 LIKE 模糊查询


  • 使用LIKE运算选择类似的值
  • 选择条件可以包含字符或数字
    %代表一个或多个字符
    _代表一个字符

    select * from emp where emp_name like '张%'


‘%’和‘-’可以同时使用

SELECT last_name
FROM employees
WHERE last_name LIKE '_o%';

1.3.4 NULL 使用NULL判断空值

select * from emp where emp_birthday is null ;

1.4 逻辑运算

1.4.1 AND

AND要求和的关系为真

SELECT employee_id, last_name, job_id, salary
FROM employees
WHERE salary >=10000 AND job_id LIKE '%MAN%';

1.4.2 OR

OR要求或关系为真

SELECT employee_id, last_name, job_id, salary
FROM employees
WHERE salary >= 10000 OR job_id LIKE '%MAN%';
