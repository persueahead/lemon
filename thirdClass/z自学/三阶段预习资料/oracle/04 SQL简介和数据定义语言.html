04 SQL简介和数据定义语言

1.SQL简介

  • SQL 是 Structured Query Language（结构化查询语言）的首字母缩写词
  • SQL 是数据库语言，Oracle使用该语言存储和检索信息
  • 通过 SQL可以实现与 Oracle 服务器的通信
  • 表是主要的数据库对象，用于存储数据
  • SQL 支持下列类别的命令

数据定义语言（DDL）: CREATE ALTER DROP
数据操纵语言（DML）: INSERT DELETE UPDATE SELECT
事务控制语言（TCL）: COMMIT ROLLBACK
数据控制语言（DCL）: GRANT REVOKE

2.数据定义语言

数据定义语言用于改变数据库结构，包括创建、更改和删除数据库对象
用于操纵表结构的数据定义语言命令有：

CREATE TABLE 创建表
ALTER TABLE 修改表
TRUNCATE TABLE 删除表中的数据 日志只记录一次 效率高
DROP TABLE 删除表

2.1创建表

create table emp
(
  emp_id NUMBER(10) not null, --员工编号
  emp_name VARCHAR2(100), --员工姓名
  emp_sex VARCHAR2(5), --员工性别
  emp_birthday DATE, --生日
  emp_salary DATE,  --薪资
  primary key (emp_id)  --主键
)

2.2修改表

alter table 表名  add constraint 字段 primary key (字段); --设置主键
alter table USER_INFO modify USER_NAME VARCHAR2(500); --修改字段长度

2.3删除表记录

TRUNCATE TABLE 表名
TRUNCATE TABLE USER_INFO

2.4删除表

DROP TABLE 表名
