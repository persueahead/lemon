1.面向对象的概念
	概念：
		将复杂的事情简单化。
		面向对象将以前的过程中的执行者，变成了指挥者。
		面向对象这种思想更符合现在人们的思考习惯的一种思想.
		以事物为中心的编程思想，程序员解决的方式，是程序的一种开发方法，将对象作为程序的基本单元。
		
	面向过程和面向对象在程序中如何体现：过程其实就是函数；对象是将函数等一些内容进行封装.		
	对事物的公共特点和行为的抽取
	对象：在现实生活中存在的具体的事物或事件
	类：对事物的公共特点和行为的抽取
	人类认识的世界：对象-->类
	在Java中：类――――》对象
	
	面向对象的特征：
		1.封装
		2.继承
		3.多态（同一个函数，多种形态）
2.如何使用Java描述对象
	类 = 属性+行为 = 成员变量+成员方法
	
	修饰符(public 类可以被任意访问)clas 类名{
		属性声明；(特征)对应类中成员变量
			变量：
			局部变量：在方法内定义的变量，只能在方法内使用
			成员变量： 属于对象所私有的 ：数据类型 变量名；
			类变量： 静态变量 所有对象拥有的一些特征
			static 数据类型  变量名 =值
		方法声明；（行为）对应类中成员方法 == 函数（）
	}
	
	万事万物皆对象
3.对象的创建以及变量的区别（局部变量 成员变量）	

	3.1 对象的创建和使用
		类名  对象名 = new 类名();  使用new +类名()创建一个新的对象
		使用： 对象名.对象的成员(成员变量、成员方法)的方式访问对象成员
		Person qiuxiang =new Person();  // 实例化
		//对象名.成员成员变量
		qiuxiang.name = "秋香";

		//对象名.成员方法()
		qiuxiang.eat("面条");
	3.2 成员变量 、局部变量、类变量
		成员变量：
			成员变量定义在类中,成员变量有默认的初始值
			成员变量在整个类中都可以被访问
			成员变量分为类成员变量和实例成员变量，实例变量存在与对象所在的堆内存中
		 	成员变量的权限修饰符可以根据需要选择一个
		局部变量：
			局部变量定义在方法中、代码块中
			局部变量没有默认的初始值，每次必须显示初始化
			局部变量存在于栈内存中,作用范围结束，变量空间会自动释放
			局部变量声明的时候不制定权限修饰符
		类变量：在类中定义  属于所有对象共有的	static int a = 1;
			  属于所有对象所共有的，只有有一个指向它的对象发生改变,则其他的对象都会跟着改变
